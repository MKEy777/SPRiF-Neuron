Layer Construction, Training, and Cost

A SPRiF layer applies Eqs. (1)–(8) independently to all neurons and emits the spike sequence [z1 , . . . , zT ]. Layers can be
stacked, and recurrence is optional rather than part of the neuron definition. Sequence-level tasks use a linear readout over
temporally aggregated hidden spikes; QTDB uses a timedistributed classifier; SHD uses a spiking readout whose
membrane evidence is accumulated after a short warm-up.
All parameters, including α, ρ, ω, G, κ, η, and λ, are optimized jointly with network weights by backpropagation
through time.
Relative to a scalar LIF neuron, SPRiF stores five state
values and adds 13 neuron-wise dynamics parameters (three
spectral parameters, two fast leaks, one coupling, six slowto-fast projection coefficients, and one reset coefficient). The
structured state updates remain O(h) for a layer of width
h; when recurrent connectivity is enabled, the shared O(h2 )
matrix multiplication dominates both LIF and SPRiF variants. We report end-to-end parameter counts to expose this
trade-off empirically.

Experiments

Benchmarks and Protocol

We evaluate whether one neuron design transfers across
five temporal regimes. S-MNIST and PS-MNIST present
784 normalized pixels sequentially, with PS-MNIST applying one fixed random permutation. QTDB is a six-class
ECG sequence-classification task. GSC uses 101-frame, 120dimensional mel-spectrogram features for 12-class keyword
spotting. SHD contains 700-channel event sequences and
20 spoken-digit classes. Accuracy is the evaluation metric
throughout.
