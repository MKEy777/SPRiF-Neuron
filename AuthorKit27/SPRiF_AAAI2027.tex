\begin{abstract}
Long-horizon temporal tasks require SNNs to retain information across extended delays while maintaining spike-based communication. However, standard spiking neurons use the same membrane state to integrate context, generate spikes, and receive post-spike reset, so each spike can disturb the temporal trace it must preserve. To address this coupling, we introduce \textbf{SPRiF} (Spectral Projective Reset Integrate-and-Fire Neuron), a spiking unit based on functional state decomposition. SPRiF separates a constrained spectral slow state, which captures real and damped rotational spectral modes without direct local reset, from a fast discharge state that generates spikes and receives a learnable projective reset. This design preserves resettable spike generation while insulating slow temporal memory from direct post-spike correction. In descriptive cross-paper comparisons across five temporal classification benchmarks, SPRiF has the highest reported mean accuracy among the listed methods on three tasks and remains competitive on the other two. Mechanism ablations identify the merged, lower-dimensional variant as the weakest configuration. On Spike-Intervention Delayed Match-to-Sample (SI-DMS), SPRiF maintains near-clean recall under intervention counts beyond the training range, while impulse-response analysis reveals heterogeneous learned temporal kernels. Together, these results support functional state decomposition as a practical neuron-level design principle for long-horizon temporal computation in SNNs.
\end{abstract}

\section{Introduction}

Spiking neural networks (SNNs) encode and transmit temporal information through discrete spike events. Their neuron dynamics must preserve relevant context while continuing to generate those events, a requirement that becomes difficult when dependencies span multiple timescales.

The leaky integrate-and-fire (LIF) neuron remains widely used because it is simple and compatible with surrogate-gradient training. Its extensions introduce adaptive variables or thresholds \cite{adlif,dalif}, resonant dynamics \cite{brf,prf,drf}, heterogeneous temporal decays \cite{bimodal,hetsyn,mtc,ltgate}, and modified reset \cite{inflor,arlif,rplif,psn}. These mechanisms enrich the available temporal responses, but the spike-generating variable usually remains both the memory carrier and the reset target.

Reset therefore alters the same recurrence that transports input history. This functional coupling, rather than limited timescale diversity alone, motivates separating temporal memory from resettable discharge without sacrificing binary communication or enlarging the network architecture.

We introduce \textbf{SPRiF} (Spectral Projective Reset Integrate-and-Fire Neuron), which implements this separation through two state groups. A slow spectral state combines one real exponential mode with a two-dimensional damped rotational mode to store temporal context. A fast discharge state projects this context to a membrane readout, emits binary spikes, and receives a learnable projective reset along $[1,\lambda]^\top$. SPRiF is trained with backpropagation through time and surrogate gradients; reset acts directly on the fast state while the slow state continues its spectral evolution.

SPRiF differs from multi-timescale models that add parallel decay channels without separating state functions. Its slow state stores temporal context, while its fast state handles spike readout and reset. The defining distinction is the absence of a direct reset path to the slow state, not simply the presence of heterogeneous time constants.

We evaluate SPRiF on five benchmarks spanning sequential images, physiological signals, speech, and event-based digits. Among the listed published results, SPRiF has the highest reported mean accuracy on S-MNIST, PS-MNIST, and QTDB and remains competitive on GSC and SHD. Its reported end-to-end parameter count is lower than that of the accuracy-leading listed baseline on S-MNIST, GSC, and SHD and close on PS-MNIST and QTDB. Mechanism ablations, recurrent spike intervention, and impulse-response analyses examine the roles of rotation, state separation, and projective reset.

The main contributions of our work are summarized below.
\begin{enumerate}
\item We formulate functional state decomposition as a neuron design in which temporal memory and spike-triggered reset occupy distinct states.
\item SPRiF instantiates this principle with a constrained spectral slow state that combines real exponential decay and damped rotational modes, together with a fast discharge state that receives a learned projective reset.
\item We evaluate SPRiF on five temporal benchmarks and probe its mechanism with three ablations, SI-DMS, and analyses of learned temporal kernels. Code will be released upon acceptance.
\end{enumerate}

\section{Related Work}

\subsection{Neuron Dynamics for Temporal Processing}

Adaptive variables, heterogeneous decays, and temporal gates broaden the response repertoire of LIF neurons \cite{asrnn,adlif,hetsyn,ltgate}. These approaches enrich temporal dynamics without imposing SPRiF's neuron-local division between spectral memory and resettable discharge.

Resonate-and-fire variants encode damped oscillations \cite{izhikevich_rf,brf,prf,drf}, while heterogeneous neurons and synapses distribute temporal decays across a network \cite{heterogeneity,hetsyn}. Compartmental models add dendritic structure and dynamics \cite{tclif,dhsnn,dendrify}; LSTM-LIF, for example, separates a dendritic long-term memory compartment from somatic discharge \cite{lstmlif}. Structured state-space dynamics provide another route to long-horizon processing \cite{silif,spikingssms}. DMP-SNN combines layer-shared, low-dimensional memory with a fast feed-forward spiking pathway \cite{sun2026dmp}. SPRiF instead places spectral memory and projective reset in distinct state groups within each neuron.

\subsection{Reset and Functional State Separation}

Existing mechanisms soften, adapt, or remove the post-spike update, and recent work also revisits reset and refractory periods through binary-activated recurrent sequence models \cite{inflor,rplif,arlif,psn,zhang2025reset}. These designs modify the spike-generating state or remove its reset. SPRiF retains reset but confines it to the fast discharge state, leaving the slow spectral state outside the local reset path. Network-level temporal processing and robustness strategies \cite{mpssnn,stochastic_gating,neuromorphic_robustness} remain complementary to this neuron-level design.

\section{Methodology}

\subsection{Limitation of Conventional LIF Dynamics}

For presynaptic activities $s_{i,t}$, a standard leaky integrate-and-fire (LIF) neuron receives input current
\begin{equation}
I_t=\sum_i w_i s_{i,t}+b,
\label{eq:lif_current}
\end{equation}
where $w_i$ is the synaptic weight of input $i$ and $b$ is a bias term. Its membrane update is
\begin{align}
v_t &= \beta v_{t-1}+I_t-z_{t-1}\theta,
\label{eq:lif_dynamics}\\
z_t &= H(v_t-\theta).
\label{eq:lif_spike}
\end{align}
Here, $\beta\in(0,1)$ is the leak factor, $H$ is the Heaviside step function, and $\theta$ is the firing threshold. The scalar $v_t$ carries temporal context, determines spike timing, and receives the preceding reset. Unrolling over $k$ steps gives
\begin{equation}
v_t=\beta^k v_{t-k}+\sum_{\ell=0}^{k-1}\beta^\ell\bigl(I_{t-\ell}-\theta z_{t-\ell-1}\bigr).
\label{eq:lif_unrolled}
\end{equation}
Input and reset histories are filtered through the same recurrence, so reset is inseparable from the state that transports temporal information. This motivates distinct memory and post-spike discharge states.

\subsection{SPRiF Spiking Neuron Model}

Figure~\ref{fig:lif_sprif_overview} contrasts the two architectures. For layer input $\mathbf{s}_t$, the input current is
\begin{equation}
I_t=W_{\mathrm{in}}\mathbf{s}_t+W_{\mathrm{rec}}\mathbf{z}_{t-1}+b,
\label{eq:input_current}
\end{equation}
where the recurrent term is omitted in feed-forward layers. In the single-neuron equations below, $I_t$ denotes the scalar current received by that neuron. Each neuron $j$ maintains a three-dimensional slow state $\mathbf{x}_t\in\mathbb{R}^{3}$ and a two-dimensional fast state $\mathbf{u}_t\in\mathbb{R}^{2}$; $\widetilde{\mathbf{u}}_t$ denotes the pre-reset fast state. Its dynamics are
\begin{equation}
\left\{
\begin{aligned}
\mathbf{x}_t &= A\mathbf{x}_{t-1}+B I_t,\\
\widetilde{u}_t^0 &= \eta_0 u_{t-1}^0+\kappa u_{t-1}^1+(G\mathbf{x}_t)^0,\\
\widetilde{u}_t^1 &= \eta_1 u_{t-1}^1+(G\mathbf{x}_t)^1,\\
v_t &= \widetilde{u}_t^0,\qquad z_t=H(v_t-\theta),\\
\mathbf{u}_t &= \widetilde{\mathbf{u}}_t-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix}.
\end{aligned}
\right.
\label{eq:sprif_dynamics}
\end{equation}
The update follows $I_t\rightarrow\mathbf{x}_t\rightarrow\widetilde{\mathbf{u}}_t\rightarrow v_t\rightarrow z_t\rightarrow\mathbf{u}_t$. The slow state is updated before spike generation; the fast state is then read out, thresholded, and reset. The direct local reset update therefore acts only on the fast path, although $W_{\mathrm{rec}}\mathbf{z}_t$ lets spikes influence the slow state at later timesteps in recurrent layers.

Equation~\eqref{eq:sprif_dynamics} is written for one neuron, with indices suppressed except in $\lambda_j$. Except for $W_{\mathrm{in}}$ and $W_{\mathrm{rec}}$, the listed dynamics parameters are per-neuron. Layers apply the update at each timestep, and recurrence is optional.

\subsection{Spectral Slow-Memory Dynamics}

The slow-state update in Eq.~\eqref{eq:sprif_dynamics} uses a three-dimensional real-valued system with one real decay mode and one complex-conjugate rotational pair. This constrained transition retains temporal context while keeping its spectrum interpretable.
\begin{equation}
A=
\begin{bmatrix}
\alpha & 0 & 0\\
0 & \rho\cos\omega & -\rho\sin\omega\\
0 & \rho\sin\omega & \rho\cos\omega
\end{bmatrix},\quad
B=
\begin{bmatrix}
1-\alpha\\ 1-\rho\\ 0
\end{bmatrix}.
\label{eq:spectral_matrix}
\end{equation}
The spectrum of $A$ is
\begin{equation}
\operatorname{eig}(A)=\left\{\alpha,\rho e^{\mathrm{i}\omega},\rho e^{-\mathrm{i}\omega}\right\}.
\label{eq:spectral_eigenvalues}
\end{equation}
The first coordinate is an exponential trace, $x_t^0=\alpha x_{t-1}^0+(1-\alpha)I_t$, while $(x_t^1,x_t^2)$ form a damped rotation driven through the first oscillatory coordinate. We parameterize
\begin{equation}
\alpha=\sigma(\alpha_{\mathrm{raw}}),\quad
\rho=\sigma(\rho_{\mathrm{raw}}),\quad
\omega=\pi\sigma(\omega_{\mathrm{raw}}),
\label{eq:parameterization}
\end{equation}
where $\sigma$ is the logistic sigmoid. Thus $\alpha,\rho\in(0,1)$ and $\omega\in(0,\pi)$ place every eigenvalue of $A$ strictly inside the unit disk. The parameters $\alpha$, $\rho$, and $\omega$ control real decay, rotational damping, and rotational frequency, respectively, yielding a stable, interpretable spectrum.

The resulting temporal basis is explicit. For a unit impulse entering the slow state, the response after $k$ subsequent transitions is
\begin{equation}
A^kB=
\begin{bmatrix}
(1-\alpha)\alpha^k\\
(1-\rho)\rho^k\cos(k\omega)\\
(1-\rho)\rho^k\sin(k\omega)
\end{bmatrix}.
\label{eq:spectral_impulse_response}
\end{equation}
The real coordinate supplies a monotone exponential trace, whereas the rotational pair supplies phase-shifted, damped oscillations. They form a compact temporal basis with distinct decay and frequency controls.

\subsection{Fast Discharge Dynamics and Projective Reset}

The fast-state equations in Eq.~\eqref{eq:sprif_dynamics} read out slow context and implement resettable discharge. The learned coefficients $\eta_0,\eta_1\in(0,1)$ are fast leaks, $\kappa$ couples the second fast coordinate to the first, and $G\in\mathbb{R}^{2\times3}$ projects slow dynamics into discharge space.

The first fast coordinate is the membrane potential $v_t=\widetilde{u}_t^0$, and the forward pass generates $z_t=H(v_t-\theta)$ with a hard threshold. During optimization, only the threshold derivative is replaced by the multi-Gaussian surrogate \cite{asrnn}; the forward dynamics remain unchanged.

After a spike, SPRiF applies the following projective reset only to the fast state.
\begin{equation}
\mathbf{u}_t=\widetilde{\mathbf{u}}_t-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix},
\label{eq:projective_reset}
\end{equation}
where $\lambda_j$ is learned independently for neuron $j$. Setting $\lambda_j=0$ recovers scalar subtraction on the membrane coordinate; otherwise, reset modifies both fast coordinates along a learned direction. Thus, $\lambda_j$ controls post-spike correction without changing the slow-state transition.

\subsection{Reset-Memory Separation Analysis}

The preceding equations separate the same-step effect of reset from the later propagation of spike signals. Relative to the pre-reset states, the reset-induced increments are
\begin{equation}
\Delta\mathbf{x}_t^{\mathrm{reset}}=\mathbf{0},\qquad
\Delta\mathbf{u}_t^{\mathrm{reset}}=-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix}.
\label{eq:reset_separation}
\end{equation}
In LIF, the reset contribution $-z_{t-1}\theta$ is accumulated through the same recurrence as input history. In SPRiF, Eq.~\eqref{eq:reset_separation} gives a zero same-step increment for $\mathbf{x}_t$ and confines reset to $\mathbf{u}_t$.

The slow-state recurrence makes this distinction explicit. Iterating the first line of Eq.~\eqref{eq:sprif_dynamics} yields
\begin{equation}
\mathbf{x}_t=A^t\mathbf{x}_0+\sum_{\tau=1}^{t}A^{t-\tau}B I_\tau.
\label{eq:slow_unrolled}
\end{equation}
Holding $I_t$ fixed, Eq.~\eqref{eq:slow_unrolled} gives $\partial\mathbf{x}_t/\partial z_t=\mathbf{0}$ for the reset operation. In recurrent layers, $\mathbf{z}_t$ can still affect $\mathbf{x}_{t+1}$ through $W_{\mathrm{rec}}\mathbf{z}_t$. The construction therefore distinguishes same-step local reset from later network communication.

\section{Experiments}

\subsection{Evaluation Questions and Protocol}

The experiments address four questions: how SPRiF compares with published temporal SNNs, which design choices contribute to its accuracy, whether additional spike-triggered updates disrupt retained context, and what temporal dynamics the trained states learn. We evaluate five benchmarks spanning sequential images (S-MNIST and PS-MNIST), physiological signals (QTDB), frame-based speech (GSC), and event-driven speech (SHD). For SPRiF, Table~\ref{tab:main_results} reports the mean and standard deviation over five seeds.

The evidence has two levels. Table~\ref{tab:main_results} provides a descriptive cross-paper comparison with recurrent SNNs using gated, adaptive, dendritic, resonate-and-fire, and other temporal neuron dynamics; their architectures and optimization protocols remain task-specific. The subsequent ablations and SI-DMS experiments use controlled variants to examine the SPRiF mechanism. The technical appendix specifies preprocessing, architectures, optimization, initialization, and reproducibility details.

\subsection{Benchmark Comparison}

SPRiF records the highest listed mean accuracy on S-MNIST, PS-MNIST, and QTDB (Table~\ref{tab:main_results}). Its absolute margins over the best listed baseline accuracies are 0.08, 0.36, and 1.43 percentage points, respectively. On GSC and SHD, SPRiF is within 0.29 and 0.18 percentage points of the listed best result. Its end-to-end parameter count is lower than that of the accuracy-leading listed baseline on S-MNIST, GSC, and SHD, and close on PS-MNIST and QTDB. Because these systems use different task-specific protocols, the comparison establishes benchmark context rather than matched statistical significance or hardware efficiency.

\subsection{Spike-Intervention Delayed Match-to-Sample (SI-DMS)}

Standard benchmark accuracy does not reveal whether additional spikes corrupt retained context. SI-DMS tests this failure mode by pairing a clean replay with a stressed replay in which forced spikes trigger native reset and recurrent propagation during the memory delay.

The test uses a 64-unit recurrent hidden layer. During the delay, 15\% of hidden neurons are induced to spike at each of $K$ sampled timesteps before a match/non-match probe. Placing the membrane coordinate above threshold before spike computation triggers native reset, and the emitted spike enters the next-step recurrent pathway. SI-DMS therefore measures combined spike-reset and recurrent-spike stress, not a pure local-reset effect. Training samples $K\in\{0,1,2,4,8\}$; evaluation adds unseen $K\in\{16,32,40\}$ and sweeps the intervention fraction from 5\% to 50\%.

For each seed and delay, $\Delta\mathrm{Acc}(K,f)=\mathrm{Acc}(0)-\mathrm{Acc}(K,f)$. Table~\ref{tab:si_dms} averages over five delays within each seed and then over three seeds at $f=15\%$. ASRNN provides the clean-matched control: full SPRiF and ASRNN both attain $100.0{\pm}0.0$ clean accuracy, but their losses at unseen $K=40$ are $0.2{\pm}0.2$ and $31.2{\pm}7.4$ percentage points, respectively. BRF and LIF start lower, with LIF also showing substantial seed variability, so clean accuracy and paired loss should be read together.

At $K=40$, the $\lambda{=}0$ variant loses $4.2{\pm}0.2$ percentage points, compared with $0.2{\pm}0.2$ for full SPRiF, supporting a task-specific benefit from the learned reset direction. Both separated variants are more resilient than the merged model, whose losses are $32.3{\pm}1.4$ at $K=8$ and $36.4{\pm}2.7$ at $K=40$, alongside lower clean accuracy. Because merging also reduces state dimensionality, this result cannot isolate separation from capacity. Figure~\ref{fig:si_dms} gives the complete delay-by-$K$ view; the appendix provides the fraction sweep and integrity audit.

Figure~\ref{fig:si_dms}a replays the same deterministic A--A input under clean and $K=8$ conditions. After the 1600-ms delay and eight forced events, the stressed replay retains natural cue-2 activity and the same correct match decision. This trace illustrates the intervention process; Fig.~\ref{fig:si_dms}b--c provides the aggregate evidence across models, delays, and seeds.

\subsection{Reset Geometry and Temporal Kernels}

The state-space diagnostics separate local dynamics from intervention behavior. The update equations give $\mathbf{x}_t$ a zero same-step reset increment, whereas SI-DMS measures recall after forced spikes undergo native reset and enter the next-step recurrent pathway. Figure~\ref{fig:mechanism}a records slow and fast states around a PS-MNIST spike, and Fig.~\ref{fig:mechanism}b follows slow-state oscillation across cue--delay probe windows. These traces visualize continuous slow-state evolution; Fig.~\ref{fig:mechanism}c--e then characterizes the learned projective reset directions.

The learned reset directions do not collapse to a common value. On QTDB, GSC, and PS-MNIST, $\lambda$ spans both signs (Fig.~\ref{fig:mechanism}c--d), and its correlations with firing rate, $\alpha$, and $\omega$ remain weak, with $|r|\leq0.191$ (Fig.~\ref{fig:mechanism}e). Combined with the $\lambda{=}0$ ablation, this variation shows that training learns heterogeneous reset directions in the fast state. The result characterizes reset geometry in the tested networks and does not imply a biological interpretation.

The impulse-response analysis complements SI-DMS by exposing the intrinsic slow transition rather than recall under induced spikes. We inject a unit impulse into trained neurons and record 100 steps of slow-state evolution without subsequent input, isolating the temporal basis $A^kB$ in Eq.~\eqref{eq:spectral_impulse_response}. Figure~\ref{fig:temporal_kernels} combines trajectories sampled across $\alpha$ quantiles with real-coordinate spectra aggregated over all neurons.

The $\alpha$-stratified real coordinates span rapid and persistent responses, and the rotational coordinates vary in phase and frequency. At the population level, the median normalized real-coordinate spectra concentrate at low frequencies, with substantial within-task variation in the 10--90\% ranges. Normalization removes absolute amplitude from this comparison. The resulting response families reflect the constrained parameterization: $\alpha$ controls monotone retention, and $\rho$ and $\omega$ govern rotational damping and phase evolution.

Available checkpoints permit the cross-model kernel comparison only on GSC and QTDB. On these tasks, selected slow-decay SPRiF real modes have derived time constants of approximately 272 and 138 steps, respectively, compared with 34 and 46 steps for the trained ASRNN membrane kernels. We treat this as a two-task diagnostic, not a general cross-model advantage. The appendix provides additional impulse-response and spectral analyses.

\section{Conclusion}

SPRiF separates temporal memory from spike-triggered discharge: a constrained spectral slow state carries real and rotational modes, and a fast state emits binary spikes and receives projective reset. Across five temporal benchmarks, SPRiF has the highest reported mean accuracy among the listed methods on three tasks and remains competitive on the other two; because the published systems use different protocols, these cross-paper results provide descriptive context. At the unseen $K=40$ setting in recurrent SI-DMS, full SPRiF loses $0.2{\pm}0.2$ percentage points, compared with $31.2{\pm}7.4$ for clean-matched ASRNN and $4.2{\pm}0.2$ for the $\lambda{=}0$ variant. The merged variant shows the largest benchmark degradation but also reduces state dimensionality, so it does not isolate separation from capacity. Under the tested conditions, functional state decomposition provides a practical way to protect temporal context, with the learned reset direction offering task-specific refinement; evaluation at larger scale and on neuromorphic hardware remains future work.
