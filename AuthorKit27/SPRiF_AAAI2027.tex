\begin{abstract}
Temporal tasks often require information to persist across long intervals. Standard spiking neurons use the same membrane state to accumulate context, generate spikes, and receive post-spike reset. This coupling repeatedly perturbs the temporal trace that a neuron must preserve. We introduce \textbf{SPRiF} (Spectral Projective Reset Integrate-and-Fire Neuron), a spiking unit based on \emph{functional state decomposition}. SPRiF separates a constrained spectral slow state, which provides real and rotational memory modes without reset, from a fast discharge state that generates spikes and receives a learnable projective reset. The design preserves a resettable spike-generation pathway while insulating slow temporal memory from post-spike correction. Across five temporal classification benchmarks, SPRiF attains the highest mean accuracy among the compared methods on three tasks and remains competitive on the other two. Ablations identify slow--fast state separation as the primary contributor, with consistent gains from spectral rotation and projective reset. A controlled spike-intervention task shows stable delay-period recall under reset stress, and impulse-response analysis reveals heterogeneous temporal kernels. These results support functional state decomposition as a practical neuron-level design principle for temporal SNNs.
\end{abstract}

\section{Introduction}

Spiking neural networks (SNNs) process temporal information through discrete spike events. They are used in speech recognition, physiological signal analysis, and event-based recognition. Unlike continuous-activation networks, their representational capacity depends on both inter-layer connectivity and neuron dynamics. These dynamics determine how neurons accumulate temporal context, decide when to fire, and update their states after each spike. When relevant information spans multiple timescales, from speech phonemes to long permuted sequences, neuron-level dynamics constrain the temporal patterns that an SNN can retain and exploit.

The leaky integrate-and-fire (LIF) neuron remains the dominant SNN unit because it is simple and compatible with surrogate-gradient training. Its extensions enrich neuronal dynamics in several ways. Adaptive-threshold models introduce slow variables that modulate firing \cite{adlif,dalif,biophysical_adaptation}. Resonant neurons add damped subthreshold oscillations \cite{brf,prf,drf}, and multi-timescale designs use heterogeneous neuronal or synaptic decays \cite{bimodal,hetsyn,mtc,ltgate}. Recent reset mechanisms also modify the post-spike update to reduce information loss \cite{inflor,arlif,rplif,psn}. Despite these advances, the spike-generating variable is usually also the reset target. In standard LIF, one membrane potential accumulates history, determines spike timing, and absorbs reset. Each spike therefore perturbs its temporal trace. Oscillatory and adaptive variants retain this membrane-centred organisation, whereas reset innovations soften, adapt, or remove reset from the same spike-generating state.

This coupling raises a basic question: \textbf{must the state that triggers a spike also store temporal memory?} A spike indicates that a discharge-related variable has crossed a threshold and therefore requires correction. It need not alter every accumulated memory trace. Insulating a memory-bearing state from spike-triggered reset could preserve a richer input history without abandoning binary communication or enlarging the network.

We address this question with \textbf{SPRiF} (Spectral Projective Reset Integrate-and-Fire Neuron), which implements \emph{functional state decomposition}. Each neuron contains a slow spectral state and a fast discharge state. The slow state integrates input through one real exponential mode and one two-dimensional damped rotational mode. It is never reset by spikes. The fast state receives a projection of the slow state, produces the membrane readout and binary spikes, and absorbs a learnable \emph{projective reset} along $[1,\lambda]^\top$. SPRiF is trained using standard backpropagation through time and surrogate gradients. Spike events therefore correct only the fast discharge state while the slow state evolves continuously.

Functional decomposition differs from conventional multi-timescale design. The latter assigns different decay rates to parallel channels, a form of \emph{rate decomposition}, while coupling their roles to readout and reset. SPRiF instead assigns distinct functions. The constrained slow state stores temporal context, and the fast state implements readout and reset. The reset operator has no update path to the slow state. The distinction is therefore structural and testable at the update level, rather than a consequence of heterogeneous time constants alone.

We evaluate SPRiF on five temporal benchmarks spanning sequential image classification, physiological signal analysis, speech recognition, and event-based digit recognition. SPRiF attains the highest compared mean accuracy on S-MNIST, PS-MNIST, and QTDB. It remains close to the strongest compared results on GSC and SHD. Its end-to-end parameter count is lower than that of the strongest compared entry on four datasets and comparable on QTDB. Ablations on PS-MNIST, GSC, and QTDB isolate the contributions of rotation, state separation, and projective reset. A controlled spike-intervention experiment and impulse-response analyses further examine the proposed dynamics.

\noindent\textbf{Contributions.}
\begin{enumerate}
\item We propose \emph{functional state decomposition} for spiking neurons, separating temporal memory storage, spike readout, and post-spike reset into distinct state components. This principle is orthogonal to rate decomposition and single-state reset.
\item We introduce a \emph{constrained spectral slow state} with real exponential decay and damped rotation modes, providing interpretable, structured temporal filtering beyond the single-exponential LIF trace.
\item We propose \emph{projective fast-state reset}, a learnable directional reset mechanism that operates exclusively on the discharge state and leaves the memory state untouched.
\item We evaluate SPRiF on five temporal benchmarks, three mechanism ablations, a controlled spike-intervention experiment, and analyses of learned temporal kernels. Code will be released upon acceptance.
\end{enumerate}

\section{Related Work}

\subsection{Neuron Dynamics for Temporal Processing}

Extensions of the leaky integrate-and-fire (LIF) neuron enrich temporal dynamics through gated integration, conductance-based modelling, adaptive thresholds, and improved surrogate-gradient training \cite{glif,dgn,asrnn,clif,radlif}. Adaptive and multi-timescale models introduce slow variables, heterogeneous neuronal or synaptic decays, conductance channels, and learnable temporal gates \cite{adlif,dalif,biophysical_adaptation,bimodal,hetsyn,mtc,ltgate}. These approaches provide \emph{rate decomposition}: parallel components evolve at different timescales, but remain coupled to membrane readout and reset.

Oscillatory and dendritic neurons offer complementary routes to temporal selectivity. Resonate-and-fire variants use damped oscillations \cite{izhikevich_rf,brf,prf,drf}, whereas compartmental and heterogeneous models introduce dendritic structure and dynamics \cite{tclif,dhsnn,dendrify,heterogeneity}. Structured state-space models have also been incorporated into spiking neurons \cite{silif,spikingssms}. SPRiF differs from these alternatives along three axes. It assigns memory and discharge to distinct state roles, confines reset to the discharge state, and constrains memory dynamics to one real and one rotational spectral mode. Its contribution is therefore a structurally separated memory--reset interface, rather than heterogeneous rates or spatial compartments alone.

\subsection{Reset and Functional State Separation}

Existing reset mechanisms soften, adapt, or remove the post-spike update to reduce information loss and, in some cases, enable parallel training \cite{inflor,rplif,arlif,revisiting_reset,psn}. They still modify the spike-generating state itself, or remove reset from that state. SPRiF instead confines a learnable projective reset to the fast discharge state and leaves the slow spectral state untouched. Network-level temporal architectures, including temporal subnetworks and stochastic spike communication \cite{mpssnn,stochastic_gating,neuromorphic_robustness}, are complementary to this neuron-level separation.

\section{Method}

\subsection{From Conventional LIF Dynamics to Functional Decomposition}

We begin with the discrete-time dynamics of a standard leaky integrate-and-fire (LIF) neuron. Given an input current $I_t$, a scalar membrane state integrates its history and then generates a binary spike,
\begin{align}
\widetilde{v}_t &= \beta v_{t-1}+I_t, &
z_t &= H(\widetilde{v}_t-\theta),
\label{eq:lif_prereset}
\end{align}
followed by the post-spike update
\begin{equation}
v_t=\widetilde{v}_t-z_t\theta,
\label{eq:lif_reset}
\end{equation}
where $\beta\in(0,1)$ is the leak factor, $H$ is the Heaviside step function, and $\theta$ is the firing threshold. This formulation is computationally attractive, but assigns three functions to the same state. The state $v_t$ retains temporal context, determines spike timing, and receives reset. Every spike therefore corrects the state that also carries neuronal memory. Without reset, the resulting memory kernel is a single exponential, proportional to $\beta^k$ after $k$ steps.

SPRiF retains the event-driven LIF interface but separates these functions. For a layer input $\mathbf{s}_t$, the current is
\begin{equation}
I_t=W_{\mathrm{in}}\mathbf{s}_t+W_{\mathrm{rec}}\mathbf{z}_{t-1}+b,
\label{eq:input_current}
\end{equation}
where the recurrent term is omitted in feed-forward layers. Rather than using $I_t$ to update one resettable membrane variable, each SPRiF neuron maintains a three-dimensional slow state $\mathbf{x}_t\in\mathbb{R}^{3}$ and a two-dimensional fast state $\mathbf{u}_t\in\mathbb{R}^{2}$. The slow state carries temporal context. The fast state converts that context into a membrane readout, spikes, and post-spike correction. The ordered computation is
\begin{equation}
I_t\rightarrow\mathbf{x}_t\rightarrow\widetilde{\mathbf{u}}_t\rightarrow v_t\rightarrow z_t\rightarrow\mathbf{u}_t,
\label{eq:sprif_flow}
\end{equation}
where $\widetilde{\mathbf{u}}_t$ denotes the pre-reset fast state. The two paths meet through a learned slow-to-fast projection, but only the fast path is reset. The following subsections specify the constrained slow memory system and the resettable discharge system.

\subsection{Constrained Spectral Slow State}

The slow state must retain context without direct spike-triggered correction and represent a richer temporal basis than the single exponential of Eq.~\eqref{eq:lif_prereset}. We therefore use the smallest real-valued linear state containing a real decay mode and a complex-conjugate rotational pair. Its update is
\begin{equation}
\mathbf{x}_t=A\mathbf{x}_{t-1}+B I_t,
\label{eq:slow_update}
\end{equation}
with
\begin{equation}
A=
\begin{bmatrix}
\alpha & 0 & 0\\
0 & \rho\cos\omega & -\rho\sin\omega\\
0 & \rho\sin\omega & \rho\cos\omega
\end{bmatrix},\quad
B=
\begin{bmatrix}
1-\alpha\\ 1-\rho\\ 0
\end{bmatrix}.
\label{eq:spectral_matrix}
\end{equation}
The spectrum of $A$ is
\begin{equation}
\operatorname{eig}(A)=\left\{\alpha,\rho e^{\mathrm{i}\omega},\rho e^{-\mathrm{i}\omega}\right\},
\label{eq:spectral_eigenvalues}
\end{equation}
so the first coordinate is an exponential trace,
$x_t^0=\alpha x_{t-1}^0+(1-\alpha)I_t$, while $(x_t^1,x_t^2)$ form a damped rotation driven through the first oscillatory coordinate. We parameterize
\begin{equation}
\alpha=\sigma(\alpha_{\mathrm{raw}}),\quad
\rho=\sigma(\rho_{\mathrm{raw}}),\quad
\omega=\pi\sigma(\omega_{\mathrm{raw}}),
\label{eq:parameterization}
\end{equation}
where $\sigma$ is the logistic sigmoid, so that $\alpha,\rho\in(0,1)$ and $\omega\in(0,\pi)$. These constraints keep all slow modes stable and retain a direct dynamical interpretation. The parameter $\alpha$ controls real decay, $\rho$ controls rotational damping, and $\omega$ controls rotational frequency. They also prevent the memory transition from becoming an unconstrained recurrent matrix.

The structure makes the temporal basis explicit. After $k$ steps, an input impulse contributes a term proportional to $(1-\alpha)\alpha^k$ through the real mode. It contributes terms proportional to $(1-\rho)\rho^k\cos(k\omega)$ and $(1-\rho)\rho^k\sin(k\omega)$ through the rotational pair. SPRiF can therefore combine monotone and oscillatory traces within each neuron while retaining a linear-time recurrence. This state is computed before spike generation and is absent from the reset equation introduced next.

\subsection{Fast Discharge State and Projective Reset}

The slow state is mapped to a two-dimensional pre-reset fast state:
\begin{align}
\widetilde{u}_t^0 &= \eta_0 u_{t-1}^0+\kappa u_{t-1}^1+(G\mathbf{x}_t)^0,\\
\widetilde{u}_t^1 &= \eta_1 u_{t-1}^1+(G\mathbf{x}_t)^1,
\label{eq:fast_update}
\end{align}
where $\eta_0,\eta_1\in(0,1)$ are learned leak coefficients, $\kappa$ couples the second fast coordinate to the first, and $G\in\mathbb{R}^{2\times3}$ projects slow dynamics into the discharge space. The first coordinate is the membrane potential, $v_t=\widetilde{u}_t^0$. The forward spike is binary:
\begin{equation}
z_t=H(v_t-\theta).
\label{eq:spike}
\end{equation}
where $H$ is the Heaviside step function and $\theta$ is the firing threshold. We train through the discontinuity with the multi-Gaussian surrogate used in recurrent SNN training \cite{asrnn}. The forward pass always uses the hard threshold.

After a spike, SPRiF corrects only the fast state:
\begin{equation}
\mathbf{u}_t=\widetilde{\mathbf{u}}_t-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix},
\label{eq:projective_reset}
\end{equation}
where $\lambda_j$ is learned independently for neuron $j$. Setting $\lambda_j=0$ recovers scalar subtraction on the membrane coordinate. Nonzero values allow reset to modify both discharge coordinates along a learned direction. Crucially, Eq.~\eqref{eq:projective_reset} contains no $\mathbf{x}_t$ term. A spike changes $\mathbf{u}_t$ but leaves the slow memory state intact. This structure preserves memory across spikes without removing reset or replacing binary communication.

\subsection{Layer Construction, Training, and Cost}

A SPRiF layer applies Eqs.~\eqref{eq:input_current}--\eqref{eq:projective_reset} independently to each neuron and emits $[\mathbf{z}_1,\ldots,\mathbf{z}_T]$. Layers can be stacked, and recurrence is optional rather than intrinsic to the neuron definition. Sequence-level tasks use a linear readout over temporally aggregated hidden spikes. QTDB uses a time-distributed classifier, and SHD uses a spiking readout whose membrane evidence is accumulated after a short warm-up. All parameters, including $\alpha$, $\rho$, $\omega$, $G$, $\kappa$, $\eta$, and $\lambda$, are jointly optimized with network weights through backpropagation in time.

Relative to a scalar LIF neuron, SPRiF stores five state values and adds 13 neuron-wise dynamics parameters: three spectral parameters, two fast leaks, one coupling, six slow-to-fast projection coefficients, and one reset coefficient. The structured state updates remain $O(h)$ for a layer of width $h$. When recurrent connectivity is enabled, the shared $O(h^2)$ matrix multiplication dominates both LIF and SPRiF variants. We report end-to-end parameter counts to quantify this trade-off.

\section{Experiments}

\subsection{Benchmarks and Protocol}

We tested whether a single neuron design transfers across five temporal regimes. S-MNIST and PS-MNIST present 784 normalized pixels sequentially, with PS-MNIST using a fixed permutation. QTDB is a six-class ECG sequence-classification task. GSC uses 101-frame, 120-dimensional mel-spectrograms for 12-class keyword spotting. SHD contains 700-channel event sequences from 20 spoken-digit classes. Accuracy is the evaluation metric throughout.

The S-MNIST and PS-MNIST networks used two recurrent SPRiF layers of widths 64 and 256. They were trained for 150 epochs with Adam, a batch size of 512, a learning rate of $10^{-2}$, and truncated BPTT chunks of 262 steps. QTDB used one recurrent layer of width 36, Adam, a batch size of 128, a learning rate of $10^{-2}$, and 250 epochs. GSC used one recurrent layer of width 300, temporal mean pooling, AdamW, a batch size of 128, a learning rate of $5\times10^{-3}$, and at most 150 epochs. SHD used a recurrent hidden layer of width 64 and a 20-neuron SPRiF readout, Adam, a batch size of 100, a learning rate of $10^{-2}$, and at most 100 epochs. Spectral parameters were initialized from log-uniform time-constant ranges and learned without clipping under the constrained parameterization. Full task-specific settings, including seed lists, are provided in the technical appendix.

We compared SPRiF with published recurrent SNNs spanning gated, adaptive, dendritic, resonate-and-fire, and recent temporal architectures. Source studies used task-specific protocols. Table~\ref{tab:main_results} therefore provides descriptive cross-paper comparisons, not significance claims. SPRiF entries report mean$\pm$std over five seeds.


\subsection{Main Results}

Table~\ref{tab:main_results} reports the main benchmark results. SPRiF attained the highest mean accuracy among the compared results on S-MNIST, PS-MNIST, and QTDB, with five-seed results of $99.28\pm0.06\%$, $95.86\pm0.14\%$, and $88.43\pm0.36\%$, respectively. The margins over the strongest available published entries were 0.08, 0.50, and 1.43 points. Because these entries used task-specific protocols, the margins are descriptive rather than cross-paper significance claims. On GSC and SHD, SPRiF reached $94.55\pm0.31\%$ and $91.52\pm0.46\%$, 0.29 and 0.18 points below the strongest compared result. Together, these comparisons show that the same neuron mechanism remained competitive across pixel sequences, physiological signals, frame-based speech, and event streams.

The comparison also indicates parameter efficiency. SPRiF used 0.067M parameters on S-MNIST and PS-MNIST, 1.77k on QTDB, 0.13M on GSC, and 0.05M on SHD. Relative to the strongest-accuracy entry for each dataset, this corresponded to approximately 55\% fewer parameters on both MNIST variants, a comparable count on QTDB (1.77k versus 1.73k), 35\% fewer on GSC, and 55\% fewer on SHD. The multi-state neuron therefore did not require a larger end-to-end network to attain the reported accuracies.

\subsection{Mechanism Ablations}

We isolated three design choices on PS-MNIST, GSC, and QTDB. Ablation A set $\omega=0$, removing rotational coupling while retaining the real decay path. Ablation B merged memory and discharge into a single three-dimensional state, whose first coordinate served as both membrane and reset target. Ablation C fixed $\lambda=0$, reducing Eq.~\eqref{eq:projective_reset} to scalar reset. All variants otherwise retained the task-specific training configuration.

Thus, the benefit of state separation extends beyond increased state dimensionality: the same three-dimensional capacity without the slow--fast split performed substantially worse. Across these settings, state separation was the largest ablated component, whereas spectral rotation and a learnable reset direction further refined temporal processing.

\subsection{Controlled Spike Intervention}

To test whether slow--fast separation protects memory under reset stress, we designed a Spike-Intervention Delayed Match-to-Sample (SI-DMS) task. A cue was followed by a delay, during which selected hidden neurons were forced to spike at controlled times. A match/non-match probe then tested recall. At each intervention timestep, 10\% of hidden neurons were forced to spike before thresholding, so that the standard reset mechanism was applied. We evaluated $K\in\{0,1,2,4,8\}$ intervention times. The maximum-stress setting, $K=8$, was not used during training. If reset reaches the memory-bearing state, these interventions should impair delay-period recall. If the slow state is structurally insulated, recall should remain more stable.

Table~\ref{tab:si_dms} reports clean ($K=0$) and maximum-stress ($K=8$) accuracy averaged over three seeds. The fully separated variant, sprif\_full, reached 92.8\% clean accuracy and lost 0.7 points under intervention stress. Removing the projective reset direction ($\lambda{=}0$; sprif\_lambda0) produced a comparable 0.6-point drop. This pattern indicates that structural separation, rather than $\lambda$ alone, was important under this protocol. The merged variant, sprif\_merged, performed near chance in both conditions (51.7\% clean and 50.4\% under maximum stress). Among the external baselines, BRF decreased from 70.4\% to 64.9\%, whereas LIF and ASRNN remained near chance throughout. Under matched SI-DMS controls, these results support the claim that functional state decomposition protects delay-period memory from spike-triggered disruption. Figure~\ref{fig:si_dms} shows the delay-by-intervention grid. Complete grids and integrity audits are provided in the appendix.

\subsection{Dynamical Analysis}

The update equations ensure that reset does not directly modify $\mathbf{x}_t$. SI-DMS tests the functional consequence of this design under controlled interventions. Figure~\ref{fig:mechanism}a shows the resulting state dynamics around recorded spikes. Figure~\ref{fig:mechanism}b shows a slow-state oscillation across probe windows in a cue--delay task. It illustrates continuous slow-state evolution, rather than providing an event-level reset test. Figure~\ref{fig:mechanism}c--e examines the learned projective reset directions.

Projective reset did not collapse to a shared direction. Across QTDB, GSC, and PS-MNIST, the learned $\lambda$ values spanned both signs (Fig.~\ref{fig:mechanism}c--d). Their correlations with firing rate, $\alpha$, and $\omega$ were weak, with $|r|\leq0.191$ (Fig.~\ref{fig:mechanism}e). Together with the $\lambda=0$ ablation loss, these observations support diverse reset directions in the fast state. They do not assign a universal biological meaning to either sign.

We next injected a unit impulse into trained neurons and recorded 100 steps of intrinsic slow-state evolution. Figure~\ref{fig:temporal_kernels} visualizes the resulting temporal basis. $\alpha$-stratified examples span rapidly decaying and persistent real traces, whereas the oscillatory coordinates vary in sign and frequency. All-neuron spectra show broad within-task variation in effective low-pass behaviour. In selected slow-decay examples, the SPRiF real modes had derived time constants of approximately 272 steps on GSC and 138 on QTDB. The closest available trained ASRNN membrane constants were 34 and 46 steps. This comparison is limited to GSC and QTDB, the only two tasks with trained ASRNN kernels in the saved analysis data. Additional impulse-response and spectral analyses are provided in the appendix.

\section{Limitations}

SPRiF replaces one scalar state with five state values per neuron. It therefore increases activation memory and neuron-local arithmetic, even when the end-to-end parameter count is small. Several baselines are drawn from their original papers under task-specific protocols. Unified reimplementations would be needed for fully controlled significance claims. The current evaluation focuses on small- to medium-scale temporal classification and does not establish energy gains on neuromorphic hardware. The diagnostic analyses demonstrate robustness under the SI-DMS intervention protocol and reveal diverse kernels, but do not establish that every learned mode is necessary for every input. These boundaries motivate matched wall-clock and spike-operation measurements, together with larger long-context benchmarks.

\section{Conclusion}

SPRiF is built on a simple principle: the state that stores temporal context need not be reset after a spike. A constrained spectral slow state supplies real and rotational memory modes. A fast discharge state produces binary spikes and receives a learnable projective reset. Across five temporal benchmarks, this design attained the highest mean accuracy in our comparison on three tasks and remained competitive on the other two with modest network size. Ablations identified slow--fast separation as the largest contributor, with additional gains from rotation and projective reset. Controlled spike-intervention and impulse-response analyses provided complementary evidence for the proposed mechanism. These findings support functional state decomposition as a reusable neuron-level design direction. Larger-scale evaluation and neuromorphic-hardware deployment remain open.
