\begin{abstract}
Temporal tasks often require information to persist across long intervals. Standard spiking neurons use the same membrane state to accumulate context, generate spikes, and receive post-spike reset. This coupling repeatedly perturbs the temporal trace that a neuron must preserve. We introduce \textbf{SPRiF} (Spectral Projective Reset Integrate-and-Fire Neuron), a spiking unit based on \emph{functional state decomposition}. SPRiF separates a constrained spectral slow state, which provides real and rotational memory modes without reset, from a fast discharge state that generates spikes and receives a learnable projective reset. The design preserves a resettable spike-generation pathway while insulating slow temporal memory from post-spike correction. Across five temporal classification benchmarks, SPRiF attains the highest mean accuracy among the compared methods on three tasks and remains competitive on the other two. Across the tested variants, ablations associate slow--fast separation with the largest performance differences, with additional gains from spectral rotation and projective reset. A controlled spike-intervention task shows stable delay-period recall under reset stress, and impulse-response analysis reveals heterogeneous temporal kernels. These results support functional state decomposition as a practical neuron-level design principle for temporal SNNs.
\end{abstract}

\section{Introduction}

Spiking neural networks (SNNs) process temporal information through discrete spike events. Their temporal capacity depends on neuron dynamics, which govern integration, firing, and post-spike updates. When relevant information spans multiple timescales, these dynamics constrain the temporal patterns that an SNN can retain and exploit.

The leaky integrate-and-fire (LIF) neuron remains widely used because it is simple and compatible with surrogate-gradient training. Its extensions introduce adaptive thresholds \cite{adlif,dalif,biophysical_adaptation}, resonant dynamics \cite{brf,prf,drf}, heterogeneous temporal decays \cite{bimodal,hetsyn,mtc,ltgate}, and modified reset \cite{inflor,arlif,rplif,psn}. Yet the spike-generating variable usually remains the reset target: a single membrane potential accumulates history, determines spike timing, and absorbs reset, so each spike perturbs its temporal trace.

Conventional LIF dynamics thus couple temporal integration, spike generation, and reset in one state; post-spike correction directly perturbs the retained input trace. Separating memory from resettable discharge could preserve temporal context without sacrificing binary communication or enlarging network architecture.

We introduce \textbf{SPRiF} (Spectral Projective Reset Integrate-and-Fire Neuron), which implements \emph{functional state decomposition}. Its slow spectral state combines one real exponential mode with a two-dimensional damped rotational mode to store temporal context without reset. Its fast discharge state projects this context to a membrane readout, emits binary spikes, and receives a learnable \emph{projective reset} along $[1,\lambda]^\top$. Trained with backpropagation through time and surrogate gradients, SPRiF confines spike correction to the fast state while the slow state evolves continuously.

Unlike conventional multi-timescale models, whose parallel channels differ in decay but remain coupled to readout and reset, SPRiF separates state functions. Its slow state stores temporal context, whereas its fast state implements readout and reset; the reset operator has no direct update path to the slow state. The distinction is structural rather than a consequence of heterogeneous time constants alone.

We evaluate SPRiF on five benchmarks spanning sequential images, physiological signals, speech, and event-based digits. SPRiF attains the highest compared mean accuracy on S-MNIST, PS-MNIST, and QTDB, and remains competitive on GSC and SHD. Its end-to-end parameter count is lower than that of the strongest compared entry on four datasets and comparable on QTDB. Ablations, controlled spike intervention, and impulse-response analyses examine rotation, state separation, and projective reset.

The main contributions of our work are summarized as follows:
\begin{enumerate}
\item We formulate \emph{functional state decomposition}, which separates temporal memory, spike readout, and post-spike reset. This principle is orthogonal to rate decomposition and single-state reset.
\item SPRiF instantiates this principle with a \emph{constrained spectral slow state} that combines real exponential decay and damped rotational modes, and a fast discharge state with a learned \emph{projective reset} that leaves memory untouched.
\item We evaluate SPRiF on five temporal benchmarks, three mechanism ablations, a controlled spike-intervention experiment, and analyses of learned temporal kernels. Code will be released upon acceptance.
\end{enumerate}

\section{Related Work}

\subsection{Neuron Dynamics for Temporal Processing}

LIF extensions enrich temporal dynamics through adaptive variables, heterogeneous decays, and temporal gates \cite{asrnn,adlif,hetsyn,ltgate}. These approaches provide \emph{rate decomposition}: parallel components evolve at different timescales but remain coupled to membrane readout and reset.

Resonate-and-fire variants use damped oscillations \cite{izhikevich_rf,brf,prf,drf}, whereas compartmental and heterogeneous models introduce dendritic structure and dynamics \cite{tclif,dhsnn,dendrify,heterogeneity}; structured state-space models have also been incorporated into spiking neurons \cite{silif,spikingssms}. In contrast, SPRiF separates memory and discharge, confines reset to discharge, and constrains memory to one real and one rotational spectral mode. It therefore provides a structurally separated memory--reset interface rather than rate or spatial decomposition.

\subsection{Reset and Functional State Separation}

Existing reset mechanisms soften, adapt, or remove the post-spike update \cite{inflor,rplif,arlif,revisiting_reset,psn}, but still modify the spike-generating state itself or remove reset from that state. SPRiF instead confines a learnable projective reset to the fast discharge state and leaves the slow spectral state untouched. Network-level temporal architectures \cite{mpssnn,stochastic_gating,neuromorphic_robustness} are complementary to this neuron-level separation.

\section{Method}

\subsection{Limitation of Conventional LIF Dynamics}

For presynaptic activities $s_{i,t}$, a standard leaky integrate-and-fire (LIF) neuron receives input current
\begin{equation}
I_t=\sum_i w_i s_{i,t}+b,
\label{eq:lif_current}
\end{equation}
where $w_i$ is the synaptic weight of input $i$ and $b$ is a bias term. Its membrane update is
\begin{align}
v_t &= \beta v_{t-1}+I_t-z_{t-1}\theta,
\label{eq:lif_dynamics}\\
z_t &= H(v_t-\theta).
\label{eq:lif_spike}
\end{align}
Here, $\beta\in(0,1)$ is the leak factor, $H$ is the Heaviside step function, and $\theta$ is the firing threshold. The scalar $v_t$ carries temporal context, determines spike timing, and receives the preceding reset. Unrolling over $k$ steps gives
\begin{equation}
v_t=\beta^k v_{t-k}+\sum_{\ell=0}^{k-1}\beta^\ell\bigl(I_{t-\ell}-\theta z_{t-\ell-1}\bigr).
\label{eq:lif_unrolled}
\end{equation}
Input and reset histories are filtered through the same recurrence, so reset is inseparable from the state that transports temporal information. This motivates distinct memory and post-spike discharge states.

\subsection{SPRiF Spiking Neuron Model}

Figure~\ref{fig:lif_sprif_overview} contrasts the two architectures. For layer input $\mathbf{s}_t$, the input current is
\begin{equation}
I_t=W_{\mathrm{in}}\mathbf{s}_t+W_{\mathrm{rec}}\mathbf{z}_{t-1}+b,
\label{eq:input_current}
\end{equation}
where the recurrent term is omitted in feed-forward layers. Each neuron $j$ maintains a three-dimensional slow state $\mathbf{x}_t\in\mathbb{R}^{3}$ and a two-dimensional fast state $\mathbf{u}_t\in\mathbb{R}^{2}$; $\widetilde{\mathbf{u}}_t$ denotes the pre-reset fast state. Its dynamics are
\begin{equation}
\left\{
\begin{aligned}
\mathbf{x}_t &= A\mathbf{x}_{t-1}+B I_t,\\
\widetilde{u}_t^0 &= \eta_0 u_{t-1}^0+\kappa u_{t-1}^1+(G\mathbf{x}_t)^0,\\
\widetilde{u}_t^1 &= \eta_1 u_{t-1}^1+(G\mathbf{x}_t)^1,\\
v_t &= \widetilde{u}_t^0,\qquad z_t=H(v_t-\theta),\\
\mathbf{u}_t &= \widetilde{\mathbf{u}}_t-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix}.
\end{aligned}
\right.
\label{eq:sprif_dynamics}
\end{equation}
The update follows $I_t\rightarrow\mathbf{x}_t\rightarrow\widetilde{\mathbf{u}}_t\rightarrow v_t\rightarrow z_t\rightarrow\mathbf{u}_t$. The slow state is updated before spike generation; the fast state is then read out, thresholded, and reset. Local reset therefore acts only on the fast path, although $W_{\mathrm{rec}}\mathbf{z}_t$ lets spikes influence the slow state at later timesteps in recurrent layers.

Equation~\eqref{eq:sprif_dynamics} is written for one neuron, with indices suppressed except in $\lambda_j$. Except for $W_{\mathrm{in}}$ and $W_{\mathrm{rec}}$, the listed dynamics parameters are per-neuron. Layers apply the update at each timestep, and recurrence is optional.

\subsection{Spectral Slow-Memory Dynamics}

The slow-state update in Eq.~\eqref{eq:sprif_dynamics} retains temporal context without direct spike-triggered correction. Rather than learning an unconstrained transition, SPRiF uses a three-dimensional real-valued system with one real decay mode and one complex-conjugate rotational pair:
\begin{equation}
A=
\begin{bmatrix}
\alpha & 0 & 0\\
0 & \rho\cos\omega & -\rho\sin\omega\\
0 & \rho\sin\omega & \rho\cos\omega
\end{bmatrix},\quad
B=
\begin{bmatrix}
1-\alpha\\ 1-\rho\\ 0
\end{bmatrix}.
\label{eq:spectral_matrix}
\end{equation}
The spectrum of $A$ is
\begin{equation}
\operatorname{eig}(A)=\left\{\alpha,\rho e^{\mathrm{i}\omega},\rho e^{-\mathrm{i}\omega}\right\},
\label{eq:spectral_eigenvalues}
\end{equation}
The first coordinate is an exponential trace, $x_t^0=\alpha x_{t-1}^0+(1-\alpha)I_t$, while $(x_t^1,x_t^2)$ form a damped rotation driven through the first oscillatory coordinate. We parameterize
\begin{equation}
\alpha=\sigma(\alpha_{\mathrm{raw}}),\quad
\rho=\sigma(\rho_{\mathrm{raw}}),\quad
\omega=\pi\sigma(\omega_{\mathrm{raw}}),
\label{eq:parameterization}
\end{equation}
where $\sigma$ is the logistic sigmoid. Thus $\alpha,\rho\in(0,1)$ and $\omega\in(0,\pi)$ place every eigenvalue of $A$ strictly inside the unit disk. The parameters $\alpha$, $\rho$, and $\omega$ control real decay, rotational damping, and rotational frequency, respectively, yielding a stable, interpretable spectrum.

The resulting temporal basis is explicit. For a unit impulse entering the slow state, the response after $k$ subsequent transitions is
\begin{equation}
A^kB=
\begin{bmatrix}
(1-\alpha)\alpha^k\\
(1-\rho)\rho^k\cos(k\omega)\\
(1-\rho)\rho^k\sin(k\omega)
\end{bmatrix}.
\label{eq:spectral_impulse_response}
\end{equation}
The real coordinate supplies a monotone exponential trace, whereas the rotational pair supplies phase-shifted, damped oscillations. Together they form a compact temporal basis with distinct decay and frequency controls. Because the slow state is computed before spikes and absent from the reset equation, its dynamics remain separate from post-spike correction.

\subsection{Fast Discharge Dynamics and Projective Reset}

The fast-state equations in Eq.~\eqref{eq:sprif_dynamics} read out slow context and implement resettable discharge. The learned coefficients $\eta_0,\eta_1\in(0,1)$ are fast leaks, $\kappa$ couples the second fast coordinate to the first, and $G\in\mathbb{R}^{2\times3}$ projects slow dynamics into discharge space.

The first fast coordinate is the membrane potential $v_t=\widetilde{u}_t^0$, and the forward pass generates $z_t=H(v_t-\theta)$ with a hard threshold. During optimization, only the threshold derivative is replaced by the multi-Gaussian surrogate \cite{asrnn}; the forward dynamics remain unchanged.

After a spike, SPRiF applies a projective reset only to the fast state:
\begin{equation}
\mathbf{u}_t=\widetilde{\mathbf{u}}_t-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix},
\label{eq:projective_reset}
\end{equation}
where $\lambda_j$ is learned independently for neuron $j$. Setting $\lambda_j=0$ recovers scalar subtraction on the membrane coordinate; otherwise, reset modifies both fast coordinates along a learned direction. Thus, $\lambda_j$ controls post-spike correction rather than the temporal memory transition and introduces no update term for $\mathbf{x}_t$.

\subsection{Reset-Memory Separation Analysis}

The preceding equations distinguish the direct effect of reset from the ordinary temporal propagation of spike signals. Relative to the pre-reset states, the reset-induced increments are
\begin{equation}
\Delta\mathbf{x}_t^{\mathrm{reset}}=\mathbf{0},\qquad
\Delta\mathbf{u}_t^{\mathrm{reset}}=-z_t\theta
\begin{bmatrix}1\\\lambda_j\end{bmatrix}.
\label{eq:reset_separation}
\end{equation}
In LIF, the reset contribution $-z_{t-1}\theta$ is accumulated through the same recurrence as input history. In SPRiF, local reset is confined to $\mathbf{u}_t$, leaving $\mathbf{x}_t$ unchanged at the same timestep.

The slow-state recurrence makes this distinction explicit. Iterating the first line of Eq.~\eqref{eq:sprif_dynamics} yields
\begin{equation}
\mathbf{x}_t=A^t\mathbf{x}_0+\sum_{\tau=1}^{t}A^{t-\tau}B I_\tau.
\label{eq:slow_unrolled}
\end{equation}
No local reset term appears in Eq.~\eqref{eq:slow_unrolled}; holding $I_t$ fixed gives $\partial\mathbf{x}_t/\partial z_t=\mathbf{0}$ for the reset operation. In recurrent layers, $\mathbf{z}_t$ affects $\mathbf{x}_{t+1}$ through $W_{\mathrm{rec}}\mathbf{z}_t$. SPRiF therefore separates local post-spike reset from ordinary network communication.

\section{Experiments}

\subsection{Benchmarks and Protocol}

We evaluate SPRiF on five temporal classification benchmarks spanning sequential images (S-MNIST and PS-MNIST), physiological signals (QTDB), frame-based speech (GSC), and event-driven speech (SHD). Accuracy is the evaluation metric throughout. These tasks differ substantially in input encoding, sequence length, and temporal structure, allowing us to test whether the proposed neuron dynamics transfer beyond a single temporal modality.

We compare SPRiF with published recurrent SNNs that use gated, adaptive, dendritic, resonate-and-fire, and other temporal neuron dynamics. The architectures and optimization protocols are adapted to each benchmark, whereas the reported SPRiF variants retain the same neuron definition. Complete dataset preprocessing, task-specific architectures, optimization settings, initialization procedures, and reproducibility details are provided in the technical appendix. Because the source studies use task-specific protocols, Table~\ref{tab:main_results} is interpreted as a descriptive cross-paper comparison rather than a significance test.


\subsection{Main Results}

Table~\ref{tab:main_results} summarizes the benchmark comparison. SPRiF obtains the highest reported accuracy among the listed methods on S-MNIST, PS-MNIST, and QTDB, while remaining competitive on GSC and SHD. The comparison covers five distinct temporal input regimes, indicating that the proposed state decomposition is not restricted to a single encoding modality. Parameter counts are reported alongside accuracy to provide architectural context; they should not be interpreted as controlled cross-paper efficiency comparisons because the published systems differ in architecture and training protocol.

\subsection{Mechanism Ablations}

We isolated three design choices on PS-MNIST, GSC, and QTDB. Ablation A set $\omega=0$, removing rotational coupling while retaining the real decay path. Ablation B merged memory and discharge into a single three-dimensional state, whose first coordinate served as both membrane and reset target. This reduced-state variant tests the consequence of collapsing memory and discharge, but does not match the full model's total state dimensionality. Ablation C fixed $\lambda=0$, reducing Eq.~\eqref{eq:projective_reset} to scalar reset. All variants otherwise retained the task-specific training configuration.

\subsection{Controlled Spike Intervention}

To test whether slow--fast separation protects memory under reset stress, we designed a Spike-Intervention Delayed Match-to-Sample (SI-DMS) task. A cue was followed by a delay, during which selected hidden neurons were forced to spike at controlled times. A match/non-match probe then tested recall. At each intervention timestep, 10\% of hidden neurons were forced to spike before thresholding, so that the standard reset mechanism was applied. We evaluated $K\in\{0,1,2,4,8\}$ intervention times. The maximum-stress setting, $K=8$, was not used during training. If reset reaches the memory-bearing state, these interventions should impair delay-period recall. If the slow state is structurally insulated, recall should remain more stable.

Table~\ref{tab:si_dms} reports clean ($K=0$) and maximum-stress ($K=8$) accuracy averaged over three seeds. The fully separated variant, SPRiF full, reached 92.8\% clean accuracy and lost 0.7 percentage points under intervention stress. Removing the projective reset direction (SPRiF $\lambda{=}0$) produced a comparable 0.6-point drop. This pattern indicates that structural separation, rather than $\lambda$ alone, was important under this protocol. The merged variant, SPRiF merged, performed near chance in both conditions (51.7\% clean and 50.4\% under maximum stress). Among the external baselines, BRF decreased from 70.4\% to 64.9\%, whereas LIF and ASRNN remained near chance throughout. The small stress losses of the two high-performing separated variants are consistent with preserved delay-period recall under this intervention protocol. Because the merged and external baselines begin near chance, their drops should not be interpreted in isolation. Figure~\ref{fig:si_dms} shows the delay-by-intervention grid. Complete grids and integrity audits are provided in the appendix.

\subsection{Dynamical Analysis}

The update equations ensure that reset does not directly modify $\mathbf{x}_t$. SI-DMS tests the functional consequence of this design under controlled interventions. Figure~\ref{fig:mechanism}a shows the resulting state dynamics around recorded spikes. Figure~\ref{fig:mechanism}b shows a slow-state oscillation across probe windows in a cue--delay task. It illustrates continuous slow-state evolution, rather than providing an event-level reset test. Figure~\ref{fig:mechanism}c--e examines the learned projective reset directions.

Projective reset did not collapse to a shared direction. Across QTDB, GSC, and PS-MNIST, the learned $\lambda$ values spanned both signs (Fig.~\ref{fig:mechanism}c--d). Their correlations with firing rate, $\alpha$, and $\omega$ were weak, with $|r|\leq0.191$ (Fig.~\ref{fig:mechanism}e). Together with the $\lambda=0$ ablation loss, these observations support diverse reset directions in the fast state. They do not assign a universal biological meaning to either sign.

To characterize learned memory kernels, we injected a unit impulse into trained neurons and recorded 100 steps of intrinsic slow-state evolution. With no subsequent input, this diagnostic isolates the temporal basis $A^kB$ in Eq.~\eqref{eq:spectral_impulse_response} from task-specific input activity. For each task, we sampled neurons across $\alpha$ quantiles to visualize representative real and rotational coordinates, and aggregated real-coordinate spectra across all neurons. Figure~\ref{fig:temporal_kernels} therefore provides both trace-level and task-wide views of the learned slow dynamics; it is a dynamical diagnostic rather than a task-level performance comparison.

Unlike SI-DMS, which probes recall under externally forced spikes, the impulse response exposes the intrinsic slow transition. Because no further input is applied after the impulse and the slow state has no local reset update, response shapes reflect the learned spectral parameters rather than subsequent post-spike correction. The real coordinate reports monotone retention, whereas the rotational coordinates expose damped phase dynamics. Thus, the analysis complements SI-DMS: the intervention task tests a functional consequence of reset isolation, while the impulse response makes the available temporal basis explicit.

At the population level, the median normalized real-coordinate spectrum summarizes the dominant spectral profile for each task, and the 10--90\% range quantifies variation across neurons. Normalization focuses this comparison on response shape rather than absolute amplitude. The quantile trajectories and spectra therefore test whether learned modes collapse to a common temporal filter or occupy a heterogeneous basis; they do not establish a task-level frequency advantage.

\subsection{Temporal-Kernel Interpretation}

Figure~\ref{fig:temporal_kernels} shows that the slow state does not implement a single decay regime. The $\alpha$-stratified real coordinates span rapid and persistent responses, while the rotational coordinates vary in phase and frequency. Task-wise spectra concentrate amplitude at low frequencies, but their 10--90\% envelopes show substantial within-task variation. The constrained spectral parameterization therefore yields an interpretable family of temporal filters rather than a fixed kernel shared by all neurons.

On GSC and QTDB, selected slow-decay SPRiF real modes had derived time constants of approximately 272 and 138 steps, respectively, compared with 34 and 46 steps for the closest available trained ASRNN membrane kernels. This comparison is limited to the two tasks with saved ASRNN kernels and is descriptive rather than controlled. Together with the state-separation and reset ablations, these observations characterize the range of learned temporal dynamics; additional impulse-response and spectral analyses are provided in the appendix.

These kernel statistics characterize temporal capacity rather than a universal performance guarantee. In particular, a longer selected real-mode time constant does not by itself establish superiority over another neuron model. Instead, the analysis makes the learned basis auditable: $\alpha$ controls monotone retention, while $\rho$ and $\omega$ govern the damping and phase evolution of rotational modes. This parameter-level interpretation connects the observed kernel diversity to the constrained slow-memory design.

\section{Conclusion}

SPRiF is built on a simple principle: the state that stores temporal context need not be reset after a spike. A constrained spectral slow state supplies real and rotational memory modes, while a fast discharge state produces binary spikes and receives a learnable projective reset. Across five temporal benchmarks, this design attained the highest mean accuracy in our comparison on three tasks and remained competitive on the other two with modest network size. Among the tested ablations, collapsing the slow and fast states produced the largest degradation, while rotation and projective reset yielded additional gains. Controlled spike-intervention and impulse-response analyses provided complementary evidence for the proposed mechanism. These findings support functional state decomposition as a reusable neuron-level design direction. Larger-scale evaluation and neuromorphic-hardware deployment remain open.
