# 慢状态轨迹分析

## 目的
用可视化直接证明"脉冲不重置慢状态"——SPRiF 的核心设计原则。

## 数据集
- PS-MNIST (784步序列，效果最直观)

## 步骤
1. 加载已训练的 SPRiF 和 LIF 模型 (PS-MNIST)
2. 取一个测试样本，逐时间步记录：
   - SPRiF: 慢状态 x_t 的三维轨迹 + 快状态 u_t 的膜电位 + 脉冲时刻
   - LIF: 膜电位轨迹 + 脉冲时刻
3. 对齐到脉冲事件 (t=0 为脉冲时刻)
4. 画对比图：
   - **上排** SPRiF 慢状态 — 脉冲时刻**连续无跳变**
   - **下排** LIF 膜电位 — 脉冲时刻**跳变归零**

## 预期
- SPRiF 慢状态跨越脉冲事件保持连续
- LIF 膜电位在每个脉冲后被重置
