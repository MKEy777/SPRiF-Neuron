# 梯度稳定性分析 — Linux 服务器运行说明

## 环境要求
- Python 3.8+
- PyTorch (CUDA 推荐)
- 依赖：`torch`, `torchvision`, `numpy`, `matplotlib`

## 分步执行

### 1. 训练 SPRiF

```bash
cd 代码/Task_S-MNIST
python train_gradient_monitor.py
```

### 2. 训练 LIF

```bash
python train_gradient_monitor.py --model lif
```

### 3. 生成所有分析图

```bash
cd 代码/experiments

python gradient_analysis/grad_norm_tracker.py                  # Exp 1-3
python gradient_analysis/temporal_grad_attenuation.py           # Exp 4 + 梯度演变图
python gradient_analysis/loss_landscape.py                      # 损失景观 3D + 等高线
python gradient_analysis/plot_gradient_figures.py               # 统合出图
```
