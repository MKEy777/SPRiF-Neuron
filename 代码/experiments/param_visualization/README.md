# 学习参数可视化 (α/ρ/ω)

## 目的
从已训练的 SPRiF 模型中提取每个神经元的 α、ρ、ω，按任务类型画直方图，验证不同任务是否学到不同的谱参数分布。

## 数据集
- PS-MNIST (长程记忆)
- QTDB (生理节律)
- GSC (短序列语音)

## 步骤
1. 加载各任务已训练的最佳 SPRiF 模型权重
2. 逐层提取 `alpha`, `rho`, `omega` 参数值
3. 画直方图/KDE，每任务一个子图
4. 不同任务用不同颜色叠加对比

## 预期
- ECG 应偏向高频 ω
- PS-MNIST 应偏向慢衰减 (小 α)
- 不同任务间分布应有明显差异

## 输出
- `param_distributions.png` — α/ρ/ω 直方图 (3列 × 1行)
- `param_per_task.csv` — 各任务各神经元参数值
